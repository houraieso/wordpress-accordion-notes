Accordion Notes for Novel

■ 小説投稿サイト向けアコーディオン形式コンテンツ管理プラグイン


■ 概要
このWordPressプラグインは、小説投稿サイトや物語系サイトで使用することを想定したアコーディオン形式のコンテンツ管理プラグインです。キャラクター設定、用語解説、世界観説明など、作品に関連する補足情報を整理して表示するのに最適です。

■ 主な機能
・柔軟なデザインカスタマイズ
・タイトル背景色・文字色の設定
・本文背景色・文字色の設定
・小タイトル文字色の設定
・角丸の調整（0-30px）
・アコーディオン間の余白調整（0-30px）
・アニメーション設定（スライド/フェード/なし）
・ドラッグ&ドロップによる項目並び替え
・階層構造に対応（親子関係のアコーディオン）
・リッチテキスト編集機能
・小タイトル（H4タグ）挿入
・太字・斜体の設定
・画像挿入機能（メディアライブラリ連携）
・レスポンシブデザイン対応

■ インストール方法
1. プラグインファイル(zip形式)をWordPressのプラグインを追加からアップロードする

2. WordPress管理画面の「プラグイン」メニューからプラグインを有効化

3. 管理画面に「Accordion Notes」メニューが追加されます

■ 使用方法
1. アコーディオンセットの作成

2. WordPress管理画面で「Accordion Notes」→「新規作成」をクリック

3. アコーディオンセットの名前を入力

4. 「アコーディオンを追加」ボタンでアコーディオン項目を追加

5. 各項目のタイトルと内容を入力

6. 必要に応じて子アコーディオンを追加（「+」ボタン）

7. スタイル設定でデザインをカスタマイズ

8. 「保存」ボタンで保存

9. ショートコードの使用
保存後に右下に表示されるショートコードを投稿や固定ページに貼り付けます：[accordion id="1"]

■ スタイル設定
右側のパネルで以下の設定が可能です：

・タイトル背景色: アコーディオンタイトル部分の背景色

・タイトル文字色: アコーディオンタイトルの文字色

・本文背景色: 展開時のコンテンツ背景色

・本文文字色: 展開時のコンテンツ文字色

・小タイトル文字色: H4タグの文字色

・角丸: ボーダーの丸み（0-30px）

・アコーディオン間の余白: 各項目間のスペース（0-30px）

・アニメーション: 展開時のアニメーション効果

■ コンテンツ編集機能
各アコーディオン項目の編集では以下の機能が利用できます：

H: 小タイトル（H4タグ）を挿入

B: 太字テキストを挿入

I: 斜体テキストを挿入

画像挿入: WordPress メディアライブラリから画像を選択・挿入

■ 階層構造の作成
「+」ボタンで子アコーディオンを追加可能

ドラッグ&ドロップで項目の順序変更が可能

無制限の階層に対応

■ トラブルシューティング

アコーディオンが表示されない
1. ショートコードのIDが正しいか確認
2. プラグインが有効化されているか確認
3. ブラウザの開発者ツールでJavaScriptエラーがないか確認

スタイルが適用されない
1. テーマのCSSとの競合がないか確認
2. キャッシュプラグインを使用している場合はキャッシュをクリア
3. ブラウザの開発者ツールでCSSが読み込まれているか確認

アニメーションが動作しない
1. jQueryが正しく読み込まれているか確認
2. 他のプラグインとのJavaScript競合がないか確認
3. ブラウザのJavaScript設定が有効になっているか確認

パフォーマンスの問題
1. 大量のアコーディオン項目（50個以上）がある場合は分割を検討
2. 画像サイズの最適化
3. キャッシュプラグインの活用

■ カスタマイズ

CSS のカスタマイズ
テーマのstyle.cssに以下のようなCSSを追加してデザインをカスタマイズできます：

/* アコーディオンコンテナ全体 */
.accordion-notes-container {
max-width: 800px;
margin: 0 auto;
}

/* タイトル部分のホバー効果をカスタマイズ */
.accordion-item-title:hover {
opacity: 0.8;
transform: translateY(-1px);
}

/* アニメーション速度の調整 */
.accordion-item-content {
transition-duration: 0.5s !important;
}


JavaScript のカスタマイズ
独自のアコーディオン動作を追加したい場合：

jQuery(document).ready(function($) {
// アコーディオンが開いた時のカスタム処理
$(document).on('click', '.accordion-item-title', function() {
// カスタム処理をここに記述
console.log('アコーディオンがクリックされました');
});
});

■ サポート
よくある質問
Q: 複数のアコーディオンセットを1つのページに表示できますか？
A: はい、異なるIDのショートコードを使用することで複数表示可能です。

Q: スマートフォンでも正常に動作しますか？
A: はい、レスポンシブデザインに対応しており、タッチ操作にも対応しています。
